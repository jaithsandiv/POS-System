POS SYSTEM - Installation Guide
Version 1.0.0
=============================================

Thank you for choosing POS System!

SYSTEM REQUIREMENTS
-------------------
- Windows 10/11 (64-bit)
- 4GB RAM minimum (8GB recommended)
- 2GB free disk space
- .NET 8.0 Desktop Runtime
- SQL Server 2019 Express or later (will be installed automatically if not present)

INSTALLATION NOTES
------------------
1. The installer will automatically detect if SQL Server Express is installed.
2. If SQL Server is not found, SQL Server 2022 Express will be installed automatically.
3. The database "POS-db" will be created and initialized with default data.
4. A default System administrator account will be created.

WHAT GETS INSTALLED
-------------------
- POS System Application
- SQL Server 2022 Express (if not already installed)
- .NET 8.0 Desktop Runtime (if not already installed)
- POS Database with default configuration

POST-INSTALLATION
-----------------
1. Launch POS System from the Start Menu or Desktop shortcut
2. Complete the business registration wizard
3. Configure your store settings, printers, and preferences
